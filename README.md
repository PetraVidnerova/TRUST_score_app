# Simple Score Calculator App

A simple web application that calculates scores from API data using the specter2 model.

## Features

- Simple user interface for entering ID numbers
- Fetches data from a public API (JSONPlaceholder)
- Uses specter2 model to calculate embeddings
- Displays calculated score to the user

## Usage

1. Enter an ID number (1-100)
2. Click "Calculate Score" or press Enter
3. View the fetched data and calculated score

## Running Locally

```bash
pip install -r requirements.txt
python app.py
```

## Deployment

This app is designed to run on Hugging Face Spaces. Simply:

2. Upload these files
3. The Space will automatically deploy with Gradio

## How it Works

1. **User Input**: User provides an ID number
2. **API Call**: Data is fetched from JSONPlaceholder API
3. **Embedding Calculation**: Text from API is processed using specter2 model
4. **Score Calculation**: A score is derived from the embeddings
5. **Display**: Results are shown to the user
